# Social Landing Page

A personal social hub website inspired by platforms like Linktree — designed to serve as a clean, centralized landing page for all of my online profiles, projects, and contact points.

This repository contains **multiple versions** of the site, preserving the original implementation while showcasing a complete modern redesign.

---

## ✨ Project Overview

This project began as an early web development exercise and has since evolved into a fully redesigned, modern UI experience.

Rather than overwriting the original work, the legacy version is preserved for reference, learning, and historical context.

**Goals of the redesign:**

- Modern, professional UI/UX
- Improved accessibility and responsiveness
- Cleaner architecture
- Enhanced performance
- Subtle animations and interactions
- Production-ready code structure
