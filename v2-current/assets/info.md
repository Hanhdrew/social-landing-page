I got you 😎 — here’s the exact modern dark palette used in that mockup (and why it works). This style is very “startup / Apple / Vercel / Linear” inspired.

🎨 Primary Color Palette (Dark Modern UI)
🌑 Background (Deep Gradient)

Top background:
#0B1220 — deep navy-black

Bottom background:
#020617 — near-black blue

👉 This creates that rich, non-flat dark look.

🧊 Card / Surface Color

Main card background:

Elevated areas / hover surfaces:
#1F2937

Borders (subtle):
rgba(255, 255, 255, 0.08)

#3B82F6

Deep accent shadow:

🧠 Why this palette looks “modern”

Because it follows current UI design rules:

✅ Very dark neutral base
✅ Slight blue tint (not pure black)
✅ Single strong accent color
✅ Muted grays for hierarchy
✅ Soft contrast (not harsh white on black)
✅ Works beautifully in OLED dark mode 🌙
